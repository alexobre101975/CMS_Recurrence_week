Recurrence semanal (Lunch)

genero una Automatizacion Usando La Recurrencia Semanal
con las Siguientes Validaciones:
1.- Informacion Completa Correctamente (Title,Description,Image etc)
2.- Activar Button Hide Print y Desactivar 
3.- Clic Recurrencia semanal 
4.- Modificar los dias que estan por default
5.- Modificar los dias y las ocurrencias
6.- Que sea grabada la recurencina semanal en la categoria de Luch
7.- Regresar y entrar a la primera captura completada anteriormente
8.- Dar clic en el boton en el boton de NO,EXIT
9.- Regresar a esa primera captura nuevamente y ahora dar clic Boton Yes, Delete This Occurrence
    y esperar que sea visible la eliminacion de esa captura solamente
10.- use Reload page to eliminate the simulation and waiting time
11.- enter the print symbol and move forward so that the capture is not visible
12.- remove lunch recurrence using JavaScript




Nota: se utilizo execute javascript para entrar a la captura.
      se utiizo Choose File para modificar los dias y las ocurrencias
      se utilizo Press Keys para dar enter en los tags
      se utilizo scroll element into view para Subir o bajar a un elemento

 

Weekly recurrence (Lunch)

Generate an Automation Using Weekly Recurrence
with the Following Validations:
1.- Complete Information Correctly (Title, Description, Image etc)
2.- Select Button Hide Print and Unselect
3.- Click Weekly recurrence
4.- Modify the days that are by default
5.- Modify the days and occurrences
6.- That the weekly recurrence be recorded in the Luch category
7.- Go back and enter the first capture previously completed
8.- Click on the NO button, EXIT
9.- Go back to that first capture again and now click Yes Button, Delete This Occurrence
     and wait for the deletion of that capture to be visible only
10.- use Reload page to eliminate the simulation and waiting time
11.- enter the print symbol and move forward so that the capture is not visible
12.- remove lunch recurrence using JavaScript





Note: execute javascript was used to enter the capture.
       Choose File was used to modify the days and occurrences
       Press Keys was used to enter the tags
       scroll element into view was used to move up or down to an element
